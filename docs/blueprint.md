# **App Name**: Astro Emporium

## Core Features:

- Product Catalog: Detailed listings of astrology-related products like rudraksh and gemstones.
- Shopping Cart: A seamless shopping cart experience for adding and managing selected products.
- Secure Checkout: A secure and encrypted checkout process for completing purchases.
- OTP Authentication: OTP-based user login/registration system.
- Personalized Recommendations: AI powered tool that suggests relevant products based on astrological profile (user input of birthdate and zodiac)
- AI Shopping Assistant: a ai powerd chat bot for shoping help and service/support

## Style Guidelines:

- Primary color: Deep purple (#4B0082), evoking spirituality and mystery.
- Background color: Desaturated purple (#E6E0EB) provides a calming backdrop.
- Accent color: Gold (#FFD700) for highlights and accents, symbolizing enlightenment.
- Headline font: 'Playfair', a modern serif for elegance; body font: 'PT Sans' for readability.
- Use celestial icons (stars, constellations) to enhance the spiritual theme.
- Clean and intuitive layout for easy browsing and product discovery.
- Subtle animations to guide the user journey, such as transitions and hover effects.
